React Job Portal

Project Overview
The React Job Portal is a simple and user-friendly web application that helps users search and explore job opportunities. Users can search for jobs, filter them by role and location, view detailed job information, and bookmark their favorite jobs.

Features

* Search jobs by keyword
* Filter jobs by role and location
* View detailed job information
* Bookmark favorite jobs
* Responsive and user-friendly interface

Technologies Used

* React JS
* JavaScript (ES6)
* HTML5
* CSS3
* React Hooks (useState)
* Git & GitHub

How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Run npm install.
4. Run npm start.
5. Open http://localhost5173 in your browser.
